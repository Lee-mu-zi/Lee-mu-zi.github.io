# Idea Http Client

## 引言

在日常开发过程中，经常会使用到Api测试工具去进行自测，常见的优秀的Api工具很多，如Apipost、Swagger、Postman、Runningman等工具，但是每个都需去来回切换工具，个人觉得是有些麻烦的，IntelliJ IDEA HTTP Client是集成在IntelliJ IDEA中的一款功能强大的HTTP请求工具。它允许开发者直接在IDE中编写HTTP请求脚本，发送请求，查看响应，并且支持保存请求历史和环境变量配置。

主要还是因为Http client

- 与Java项目代码同处一个环境
- `.http`文件可以作为项目的Api文档
- HTTP文件可纳入版本控制
- 不用在工具之间来回切换
- 能够直接使用别人书写好的接口测试
